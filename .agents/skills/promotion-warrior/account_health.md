# 账号健康评分 — account_health.md

> 账号健康是连续变量，不是简单的 active/banned 二元状态。
> Agent 每次操作后更新健康分，分数下降时自动降低操作强度，提前发现封号风险。

---

## 健康分计算规则

**满分 100 分，分数越高越健康。**

### 加分项（每次操作后检查）

| 事件 | 加分 |
|------|------|
| 评论/帖子获得点赞 | +0.5 |
| 评论获得回复 | +1 |
| 私信获得回复 | +2 |
| 被其他用户主动关注 | +1.5 |
| 帖子被收藏/转发 | +1 |
| 操作执行后无任何警告 | +0.2 |

### 减分项（自动检测）

| 事件 | 减分 |
|------|------|
| 评论被平台折叠/隐藏 | -5 |
| 内容被删除 | -8 |
| 收到平台警告提示 | -10 |
| 出现验证码 | -3 |
| 出现操作频率限制提示 | -5 |
| 私信被标记为垃圾 | -8 |
| 粉丝数异常下降（>5%/天） | -10 |
| 互动率连续 3 天下降 | -3 |
| 连续 3 条评论无任何互动 | -1 |

### 时间自然恢复

```
每过 24 小时无减分事件：+1（最高恢复至 95 分）
```

## 健康分对应操作策略

| 健康分 | 状态标签 | 操作策略 |
|--------|----------|----------|
| 90-100 | 🟢 优秀 | 正常操作，可以参与协同打法 |
| 75-89 | 🟡 良好 | 正常操作，暂停协同打法 |
| 60-74 | 🟠 警戒 | 操作量减半，只做 Tier 1，暂停私信 |
| 40-59 | 🔴 危险 | 操作量降至 20%，只做点赞和浏览，不发任何文字 |
| 20-39 | ⛔ 休眠 | 停止所有主动操作，只保持每天登录浏览 |
| 0-19 | 💀 放弃 | 停止使用，通知用户考虑是否注销重开 |

