## 账号与人设绑定

每个账号在 `accounts.md` 中有 `persona` 字段，绑定后不要随意更换，保持风格一致。

## 新增人设

复制任一现有文件，修改以下内容：
1. 基本设定（身份、标签、语气）
2. 句式库（Tier 1/2/3 各至少 5 条）
3. 禁止行为列表

然后在 `accounts.md` 中给对应账号设置新人设文件名即可。
