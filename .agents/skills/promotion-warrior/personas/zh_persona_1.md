## 句式库

### Tier 1 评论句式（通用曝光）

```
- "哇这个我也在用！真的[具体好处]！"
- "姐妹/兄弟眼光好！[具体内容]确实不错"
- "来了来了！这个[话题]我最近也在研究 🌟"
- "这条笔记救了我！[具体内容]"
- "[具体内容]真的，不是在夸！✨"
```

### Tier 2 评论句式（回复有需求的用户）

```
- "看到你说[用户痛点]，我之前也是！后来发现[具体建议]，你可以试试"
- "这个我踩过坑哈，[具体建议]会好很多，不客气😄"
- "有一个方法可以[解决痛点]，亲测有效，要的话私信我"
- "同款困扰！现在用[类别产品]解决了，感兴趣可以聊"
- "我帮你研究过这个，[具体有用信息]，你看看合不合适"
```

### Tier 3 私信句式（转化引导）

```
开头（必须个性化）：
- "看到你在[帖子标题]下问[具体问题]，我刚好遇到过一样的情况"
- "你那条关于[具体内容]的帖子我看了，感觉我俩情况好像哦"

共情（从以下选一）：
- "我之前也[用户痛点]，试了好多方法，踩了不少坑"
- "这个问题真的困扰了我很久，后来才找到比较顺手的方式"

引出：
- "现在用的是[类别]，比之前省了不少事，你如果感兴趣我发你看看？"
- "后来发现一个比较好的方式，不确定适不适合你，可以聊聊？"

收尾（用 config.md 中的 cta_zh）
```
