## 回复生成规则

**回复原则：**
- 必须在 6 小时内回复 P0/P1（超过 24h 回复率大幅下降）
- 回复内容必须针对对方的具体内容个性化，不能用模板直接发
- 每次回复都要推进漏斗：Tier 1 回复 → 引导到 Tier 2；Tier 2 回复 → 发私信

**P1 回复后的标准动作：**
1. 在评论区简短回复（感谢 + 一句话补充）
2. 同时发私信，开头引用刚才的评论互动
3. 在 promotion_log.md 更新状态为"已升级 Tier 3"

**P0 私信回复结构：**
```
[对方消息的直接回应，不要说"你好"]
[补充有用信息或解答疑问]
[自然引出下一步：发链接 / 继续聊 / 给更多细节]
```

## 扫描计划

Agent 每次执行时按以下顺序扫描回复：

1. 小红书：进入"消息"→"评论和@"，找新回复
2. 微博：进入"通知"，找评论回复和私信
3. 抖音：进入"消息"，找评论回复
4. Instagram：进入"通知"，找评论回复和 DM
5. X：进入"通知"→"提及"，找回复；进入 DM 查新消息
6. Facebook：进入"通知"，找 Group 帖子回复和 Messenger
7. Reddit：进入"Inbox"，找评论回复和私信
8. TikTok：进入"收件箱"，找评论回复
