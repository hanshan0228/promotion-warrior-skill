# 跨平台用户去重 — cross_platform_users.md

> 同一个真实用户可能在多个平台都活跃。
> 如果我们在多个平台接触同一个人，对方很可能意识到被跟踪，进而举报。
> 本文件追踪疑似同一用户的跨平台账号，避免重复接触。

---

## 识别规则

以下特征高度重合时，判定为疑似同一用户：

```
强信号（任意 2 条 → 判定为同一人，停止在其他平台接触）：
  - 用户名完全相同或仅有平台后缀差异（@user vs user_xhs）
  - 头像完全相同
  - 简介文字高度相似（> 80%）
  - 在不同平台发布了相同或高度相似的内容

弱信号（需要 3 条以上才判定）：
  - 用户名相似但不完全相同
  - 发帖时间规律相同（如都在每天晚上 9-11 点发）
  - 内容风格和用词习惯高度相似
  - 关注的账号有大量重叠
```

## 执行规则

```
每次找到目标用户后，在操作前：
  1. 提取用户名/头像/简介关键词
  2. 与 cross_platform_users.md 中的已知账号对比
  3. 与 promotion_log.md 中的已接触用户对比
  4. 发现强信号匹配 → 跳过此用户，不做任何操作
  5. 发现弱信号匹配 → 记录到"疑似"区域，本次跳过

新发现跨平台关联时：
  → 更新此文件，将新平台账号加入对应用户的 known_accounts
  → 在 promotion_log.md 中标记相关记录
  → 如该用户正在 conversation_tracker.md 中跟进，保留跟进，但其他平台停止接触
```
