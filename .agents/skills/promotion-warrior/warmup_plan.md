# 养号计划 — warmup_plan.md

> 新账号必须严格按此计划执行，不可跳过阶段。
> Agent 根据 accounts.md 中的"注册天数"字段自动判断当前阶段。

---

## 通用原则

- 养号期间**绝对不发任何推广相关内容**，包括暗示
- 每天操作时间固定在同一时段（模拟真人习惯）
- 操作间隔随机化，不要机械地每隔固定秒数操作
- 养号期间账号头像、简介、主页必须提前填写完整

## 中文平台养号计划（小红书 / 微博 / 抖音）

### 第 1-3 天：纯消费行为
```
目标：让平台认为你是真实新用户在探索内容
操作：
  - 每天浏览 20-30 分钟（用 openclaw browser 模拟滚动）
  - 对感兴趣的内容点赞 10-15 次（与 niche 相关）
  - 关注 5-8 个同类博主
  - 不发任何评论，不私信任何人
```

### 第 4-7 天：开始互动
```
目标：建立账号互动记录
操作：
  - 继续每天点赞 10-15 次
  - 开始发 Tier 1 评论，每天 3-5 条（纯真实互动，不涉及任何产品）
  - 评论内容：对帖子内容的真实反应，不引导，不暗示
  - 关注账号增加到累计 20-30 个
  小红书额外：发 1-2 条与 niche 相关的真实笔记（无推广）
  微博额外：转发 2-3 条相关内容并加短评
```

### 第 8-14 天：提升权重
```
目标：账号被平台算法认定为活跃真实用户
操作：
  - Tier 1 评论增加到每天 8-10 条
  - 开始少量 Tier 2（每天 1-2 条，仍不提产品，只给有用建议）
  - 收藏内容 5-10 条/天
  - 小红书：再发 2-3 条笔记，开始在笔记里建立人设形象
  - 微博：发 1-2 条原创内容/天（与 niche 相关的干货）
  - 抖音：对视频完整观看（不跳过），点赞+评论配合
```

### 第 15 天起：正式启动
```
条件满足才能启动完整的 Tier 1/2/3：
  小红书：粉丝 > 10，笔记 > 3 条，评论记录 > 20 条
  微博：粉丝 > 5，发帖 > 5 条，互动记录 > 30 条
  抖音：关注 > 20，点赞视频 > 50，评论记录 > 15 条
```

## 英文平台养号计划（Instagram / X / TikTok）

### 第 1-3 天：完善主页
```
目标：账号看起来真实可信
操作：
  - 上传头像（真实人物照片风格，非卡通）
  - 写简介（符合人设，不含推广词）
  - Instagram：发 3-5 张生活类图片（与 niche 相关但不推广）
  - X：发 3-5 条日常推文（观点、转发评论）
  - TikTok：点赞 20+ 个视频，关注 10+ 个账号
  - 不评论，不私信
```

### 第 4-7 天：建立互动记录
```
操作：
  - 每天点赞 20-30 次
  - 关注账号 10-15 个/天（与 niche 相关）
  - 开始 Tier 1 评论，3-5 条/天（纯真实互动）
  - Instagram：继续发内容 1 条/天
  - X：每天发 2-3 条推文 + 转发 2-3 条
```

### 第 8-14 天：提升活跃度
```
操作：
  - Tier 1 评论 8-10 条/天
  - 开始少量 Tier 2（1-2 条/天，不提产品）
  - Instagram：开始用 Story 功能（投票、问答等互动功能）
  - X：参与热门话题讨论，积累曝光
  - TikTok：完整观看视频（提升账号权重信号）
```

### 第 15 天起：正式启动
```
条件：
  Instagram：粉丝 > 20，帖子 > 8，评论记录 > 30
  X：粉丝 > 10，推文 > 15，互动记录 > 40
  TikTok：关注 > 30，点赞视频 > 100，评论 > 20
```

## Facebook 养号计划

### 第 1-7 天
```
- 完善个人资料（头像、简介、工作经历等，越完整越好）
- 加入 5-10 个相关 Group
- 在 Group 里只点赞，不发言
- 添加 5-10 个好友（真实认识的人优先）
```

### 第 8-21 天
```
- 在 Group 里开始发言（每天 1-2 条，纯价值内容）
- 分享外部文章（与 niche 相关）
- 对他人帖子发表有价值的评论
```

### 第 22 天起：正式启动
```
条件：每个目标 Group 内发言记录 > 10 条，Group 成员认可度好
```

## Reddit 养号计划（最严格）

### 第 1-30 天：纯 karma 积累
```
策略：在非推广 subreddit 积累 karma
  - 回答问题（r/AskReddit, r/explainlikeimfive 等高流量版块）
  - 每天 3-5 条高质量回答
  - 不在目标 subreddit 发任何内容
  - 目标：karma 达到 100+
```

### 第 31-60 天：进入目标 subreddit
```
- 开始在目标 subreddit 发 Tier 1（纯价值，绝不提产品）
- 每天 1-2 条，间隔 > 4 小时
- 目标：在目标 subreddit 建立正向 karma 记录
```

### 第 61 天起：可以开始 Tier 2
```
条件：总 karma > 200，目标 subreddit karma > 50
```

## 养号状态更新规则

Agent 每天执行时，自动检查 accounts.md 中账号的注册天数：
- 天数 < 7 → `status: warming`，只执行对应阶段任务
- 天数 7-14 → `status: warming`，执行提升权重阶段
- 满足条件 → 自动将 `status` 更新为 `active`，通知用户
