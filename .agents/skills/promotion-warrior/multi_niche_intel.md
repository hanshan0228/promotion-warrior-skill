# Multi-Niche Intel Collector

## 1. Cursor (AI IDE)
*   **Target Competitor**: GitHub Copilot, VS Code
*   **Weakness (Copilot)**: Lazy completions, forgets context, can't rewrite multiple files at once.
*   **Pivot Point (Sniping)**: "Copilot is fine for autocomplete, but if you need to refactor a whole component, Cursor's 'Composer' feature is miles ahead because it indexes your entire codebase."
*   **High-Value Audience**: CTOs, Senior Devs, Agency Founders.

## 2. Clay (B2B Data)
*   **Target Competitor**: Apollo.io, ZoomInfo, Lusha
*   **Weakness (Apollo)**: Static databases are 30% outdated. Everyone uses the same lists, leading to spam.
*   **Pivot Point (Sniping)**: "Apollo is great for a raw list, but you're emailing the exact same people as your competitors. Using Clay for 'Waterfall Enrichment' ensures you only email people who actually match your ICP in real-time."
*   **High-Value Audience**: Head of Sales, SDR Managers, B2B Founders.
